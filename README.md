# Captured by Cadence
## Project Description
Captured by Cadence is a photography portfolio website designed to showcase my work through a clean and elegant interface. The website highlights my photography style, creative vision, and selected work while encouraging visitors to explore my portfolio and connect with me. The overall theme centers around the tagline, "Capturing the Essence of a Moment."

---

## Intended User or Player
This interface is designed for potential photography clients, creative collaborators, brands, and anyone interested in viewing my photography portfolio.

---

## User / Player Goal
Users should be able to:
- Learn about me as a photographer.
- Browse a selection of my photography.
- Understand the types of photography I specialize in.
- Easily navigate the website.
- Find a way to connect with me through my Instagram.

## Design Choices

### Layout
A single-page scrolling portfolio with clear sections for the hero image, about, selected work, photography focus, and contact information.

### Color
Muted earthy tones including warm cream, soft beige, taupe, and charcoal to create a fancy look.

### Typography
Cormorant Garamond is used for headings to create an elegant editorial feel, while Inter is used for body text to improve readability.

### Visual hierarchy
Large photos and headings draw attention, followed by text and clear call-to-action buttons.

### Feedback states
Navigation links, gallery images, and buttons include hover effects to provide visual feedback when users interact with them.

### Responsive design
A media query adjusts the navigation and image gallery for smaller screens, allowing the website to be readable and functional on different devices.
